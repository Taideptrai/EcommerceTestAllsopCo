
# B2B Food Service App
A reactive Single-Page with some new features!! 

# Usage
1. Fork the repo and then clone it or download it.

2. First install all dependencies:
    ```bash
    # with npm
    npm install
3. Start the server
    ```javascript
    npm start 
    ```
